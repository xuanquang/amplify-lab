Chuẩn bị

Kịch bản

Trong bài lab này, chúng ta sẽ bắt đầu bằng cách tạo một ứng dụng React mới và khai báo (submitting) nó cho một GitHub repository (repository, viết tắt repo, tạm dịch kho lưu trữ). Kho lưu trữ này sẽ được liên kết với dịch vụ lưu trữ web AWS Amplify và được triển khai tới mạng phân phối nội dung có thể truy cập toàn cầu (CDN, Content Delivery Network) được lưu trữ trên tên miền amplifyapp.com. Sau đó, chúng ta sẽ thể hiện khả năng triển khai liên tục (continuous deployment) bằng cách thực hiện các thay đổi đối với ứng dụng React và đẩy một phiên bản mới cho nhánh chính, phiên bản mới này sẽ tự động bắt đầu một triển khai mới.

Những gì bạn sẽ đạt được

Ở bài lab này, bạn sẽ:
- Xây dựng một ứng dụng React
- Tạo một kho lưu trữ GitHub
- Dùng AWS Amplify để triển khai ứng dụng của bạn
- Thực hiện thay đổi mã (code) và triển khai lại ứng dụng của bạn.

Chuẩn bị

  1. Làm lab với trình duyệt Chrome hoặc Firefox.

  2. Tài khoản AWS của bạn.

  3. Tải và cài đặt Visual Studio Code

  4. Tạo tài khoản Github

  5. Cài đặt Git trên máy tính của bạn và thêm SSH keys to your GitHub account.

  6. Tải và cài đặt NodeJS

Tạo một ứng dụng React mới

  1. Khởi chạy Visual Studio Code (VSC).
    • Đăng nhập vào VSC bằng tài khoản GitHub của bạn.
    • Chọn New Terminal từ menu Terminal.
    • Dùng câu lệnh (Ứng dụng React mới được gọi là aws-react-hosting):
npx create-react-app aws-react-hosting

  1. Dự án đã được tạo nên bây giờ chúng ta hãy mở thư mục trong VSC.
cd aws-react-hosting
  1. Để chạy ứng dụng React, gõ câu lệnh:
npm start

Nó đã được thiết lập và chạy trên localhost (localhost:3000)

Bước tiếp theo là tạo một Github repo mới.

Tạo Github repo mới

  1. Đi đến Create a new repository

Chúng ta cần đặt tên cho kho lưu trữ này, ta gọi là aws-react-hosting giống như tên thư mục. Nhấn vào Create repository.

Kho lưu trữ GitHub đã được tạo. Đây là một số câu lệnh mà chúng ta có thể sử dụng.

Tải ứng dụng React Native của bạn lên kho lưu trữ Github

  1. Quay trở lại VSC. VSC đang điều hướng đến thư mục React (thư mục aws-react-hosting) Hãy thử mở một cửa sổ terminal mới:
    - Để mở cửa sổ terminal mới, nhấn vào + Developer Command Prompt trong cửa sổ terminal.
    - Như chúng ta thấy, VSC có hai cửa sổ terminal: node đang chạy ứng dụng React và bash chúng ta vừa mới tạo.

  2. Chúng ta muốn đưa toàn bộ mã cần thiết đang lưu trên máy tính lên Github repo. Để làm vậy, bạn chỉ cần chạy các lệnh:

git init
git add .
git commit -m "added react app"
git branch -M main
git remote add origin <https://..... git>
git push -u origin main

URL endpoint <https://….. git>: bạn sao chép nó từ GitHub repo của bạn.

Nó sẽ đẩy tất cả các files vào Github repo. Đã xong rồi.

Quay trở lại Github repo của bạn. Refresh và bạn có thể thấy rằng tất cả các files quan trọng đều đã được tải lên ngoại trừ các node modules .
